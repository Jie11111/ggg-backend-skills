## 3. 处理链路

### 3.1 核心处理步骤

1. 参数校验、鉴权、幂等判断。
2. 加载运行时对象、历史数据或上游返回。
3. 执行业务规则、状态流转和数据组装。
4. 调用关键下游依赖并完成数据读写。
5. 组装返回。

### 3.3 接口流程图

```plantuml

:参数校验 / 鉴权;
if (校验通过?) then (是)
  :加载业务对象 / 历史数据;
  :执行业务规则;
  if (需要调用下游?) then (是)
    :发起 Dubbo / HTTP 调用;
    :接收返回;
  endif
  :写入数据 / 组装返回;
else (否)
  :返回错误;
endif
stop
@enduml
```
